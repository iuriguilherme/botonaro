Metamemo

METAMEMO é um sistema criado para coletar, armazenar e visualizar as memórias de qualquer pessoa (ou organização, coletivo, empresa, marca) nas mais populares plataformas de redes sociais da internet. O objetivo é preservar e facilitar a análise desse conteúdo por parte de ativistas, jornalistas e pesquisadores.

Um METAMEMO é um arquivo contendo um arquivo das redes sociais e suas respectivas mídias agregadas de uma pessoa / coletivo / empresa / marca.

Botonaro

Botonaro é um chatbot atualmente disponível como bot de Telegram que é programado para responder mensagens como se fosse um Bolsonaro. Ele lê o que é dito, pesquisa no banco de dados Bolsodata e escolhe uma frase aleatória para dar como resposta.

O bot de Telegram já está no ar e funciona como um sistema de busca de Metamemos do Bolsodata. Está em desenvolvimento esta mesma funcionalidade por intermédio de outras plataformas de mensagem instantânea que permitam chatbots (Discord, Matrix, Twitter, Whatsapp, etc.) - acompanhe a parte de anúncios no Github para ser avisada(o) quando estiver disponível para mais plataformas, ou ajude / opine no desenvolvimento!

Em breve estarão disponíveis neste site alguns dados de estudos de ciência dos dados, com gráficos e análise das reações dos usuários às mensagens do Botonaro (dados anonimizados).

Assim que os dados do Bolsodata forem disponibilizados aos desenvoledores e bolsistas, está prevista uma atualização no sistema do Botonaro (que é mais complexo do que um simples chatbot) para criar um vocabulário e responder de forma criativa as pessoas, efetivamente dando "vida proópria", dentro do conceito de inteligência artificial, ao Botonaro. Isto deve tornar as interações mais interessantes.
